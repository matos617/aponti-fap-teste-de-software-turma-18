# Exercício 03 - Organização do README
## A estrutura básica para um README profissional deve conter: 
- Título do projeto;
- Objetivo;
- Tecnologias utilizadas;
- Estrutura das pastas;
- Como executar o projeto;
- Autor.

Exemplo:
---

# Sistema de Gerenciamento de Tarefas
## :bookmark_tabs: Objetivo 
Este projeto tem como objetivo desenvolver um sistema simples para gerenciar tarefas, permitindo adicionar, editar e remover atividades.

## :computer: Tecnologias utilizadas 
- Git e GitHub
- Markdown
- Python 3.10
- Visual Studio Code

## :card_index_dividers: Estrutura das pastas 
```plaintext
projeto/
│── src/              # Código-fonte
│   └── main.py
│── docs/             # Documentação
│   └── manual.pdf
│── imagens/          # Recursos visuais
│   └── exemplo.gif
│── README.md
```

## :keyboard: Como executar o projeto 
1. Clone este repositório:
   ```bash
   git clone https://github.com/usuario/projeto.git
   ```
2. Acesse a pasta do projeto:
   ```bash
   cd projeto
   ```
3. Execute o arquivo principal:
   ```python
   python src/main.py
   ```
