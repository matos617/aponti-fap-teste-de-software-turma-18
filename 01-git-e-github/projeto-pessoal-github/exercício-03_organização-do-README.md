# Exercício 03 - Organização do README
## A etrutura básica para um README profissional deve conter: 
- Título do projeto;
- Objetivo;
- Tecnologias utilizadas;
- Estrutura das pastas;
- Como executar o projeto;
- Autor.

Exemplo:
---

# Sistema de Gerenciamento de Tarefas
## Objetivo
Este projeto tem como objetivo desenvolver um sistema simples para gerenciar tarefas, permitindo adicionar, editar e remover atividades.

## Tecnologias utilizadas
- Git e GitHub
- Markdown
- Python 3.10
- Visual Studio Code

## Estrutura das pastas

projeto/
│── src/              # Código-fonte
│   └── main.py
│── docs/             # Documentação
│   └── manual.pdf
│── imagens/          # Recursos visuais
│   └── exemplo.gif
│── README.md

## Como executar o projeto
1. Clone este repositório:
   ```bash
   git clone https://github.com/usuario/projeto.git
```
