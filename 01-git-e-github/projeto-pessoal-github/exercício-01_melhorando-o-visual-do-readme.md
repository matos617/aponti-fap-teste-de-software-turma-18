# Melhorando o visual do README
Uso da linguagem do Markdown para um melhor visual do seu repositório no GitHub.

## Títulos e Subtítulos
Hierarquia/Sintaxe:
```
# Título 1
## Título 2
### Título 3 
#### Título 4
##### Título 5
###### Título 6
```
Demonstração:
# Título 1
## Título 2
### Título 3 
#### Título 4
##### Título 5
###### Título 6
