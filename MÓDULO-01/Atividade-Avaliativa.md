# Atividade
## O que é teste de software?
Teste de Softtware é uma área da Engenharia de Software onde são utilizados e planejados testes nas aplicações para encontrar erros, garantir a qualidade e prevenir custos de manutenção. 

## Por que testar é importante para o negócio?
É necessário testar o software para garantir a sua qualidade e eficiência, além de confirmar os requisitos pré-estabelecidos no desenvolvimento do produto, para que não haja erros que custem na manutenção no futuro, uma melhor organização do investimento do produto final e para o negócio de quem produz/usa.

## O que caracteriza um software com qualidade?
O sentido de qualidade é muito abstrato, mas na maioria das definições se enquadra como um Produto de Qualidade o software que atenda aos requisitos do cliente ou até se supere dos requisitos mínimos atribuídos aos objeivos do desenvolvimento, tenha boa usabilidade, fácil de manutenção futura, confiabilidade, etc.

## Qual o papel do QA em um time moderno
A Garantia da Qualidade do software é essêncial no desenvolvimento do software. Ela participa desde o a fase de início do desenvolvimento do produto, incluindo aspectos funcionais e atributos de desempenho.
